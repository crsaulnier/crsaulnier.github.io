Passion Project: https://ice.report

Skills/Technologies: Python Web Development (Plotly Dash, Flask), NoSQL Database (DynamoDB), Data Science (Pandas), CI/CD DevOps (Docker, Github Actions, AWS Serverless Deployments)


As an engineer and ice climber living in Boston, I got tired of obsessively watching weather forecasts to decide where to climb on the weekends. ice.report weaves together historical weather observations and forecasts to create a single stop for ice conditions forcasting.

This project was also an excuse to play around with some technologies that seemed useful. The site is written using Plotly Dash and deployed serverlessly on AWS Lambda using Zappa. This is a little slower but lets me keep the cost of hosting the site at around a dollar a month.
